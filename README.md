# Raycast Codex Auth

A Raycast extension for quickly switching between multiple Codex accounts.

```sh
npx rayext install jinghaihan/raycast-codex-auth
```

This project is built on top of [`@loongphy/codex-auth`](https://www.npmjs.com/package/@loongphy/codex-auth).
Instead of re-implementing authentication logic, the extension delegates account switching to the same `codex-auth` CLI flow.

## Command

<!-- commands -->

| Title                  | Description                    |
| ---------------------- | ------------------------------ |
| `Switch Codex Account` | List and switch Codex accounts |
